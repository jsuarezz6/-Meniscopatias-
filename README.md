# 🩺 FisioMed-Interna

## Sistema Inteligente de Evaluación Fisioterapéutica

**Versión:** 1.0.0

---

## Descripción

FisioMed-Interna es un software clínico desarrollado para apoyar el proceso de evaluación fisioterapéutica en pacientes con patologías musculoesqueléticas.

La primera versión está diseñada para la evaluación completa de pacientes con **meniscopatías de rodilla**, integrando herramientas clínicas, escalas funcionales, pruebas semiológicas, razonamiento fisioterapéutico, diagnóstico y pronóstico automatizados.

---

# Objetivos

- Digitalizar la historia clínica fisioterapéutica.
- Automatizar cálculos clínicos.
- Mejorar el razonamiento fisioterapéutico.
- Generar diagnósticos fisioterapéuticos.
- Emitir pronósticos automáticos.
- Exportar informes clínicos.
